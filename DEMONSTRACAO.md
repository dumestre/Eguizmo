# Demonstração do egui-gizmo

Este projeto demonstra o uso da biblioteca `egui-gizmo` para criar gizmos 3D em aplicações egui.

## Como Usar

1. Certifique-se de ter o Rust instalado (versão 1.76 ou superior)
2. Clone este repositório: `git clone https://github.com/dumestre/Eguizmo`
3. Navegue até o diretório: `cd Eguizmo`
4. Execute o exemplo: `cargo run --example simple_demo`

## Estrutura do Projeto

- `src/` - Código-fonte da biblioteca egui-gizmo
- `examples/` - Exemplos de uso da biblioteca
- `demo/` - Demonstração mais completa usando Bevy
- `DOCUMENTACAO_USO.md` - Documentação detalhada de uso
- `INSTRUCOES_USO_PROJETO.md` - Instruções específicas para integração em projetos

## Funcionalidades

- Gizmos 3D para rotação, translação e escala
- Modos global e local
- Personalização de cores e aparência
- Sistema de snapping opcional
- Compatibilidade com diferentes bibliotecas de matemática via mint
